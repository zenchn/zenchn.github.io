# VSCode 快捷键

# 编辑相关

快捷键 说明
option + 箭头 上下移动行
option + shift + 箭头 上下复制行
command + 箭头 移动到行首尾
command + enter 向下重开一行
command + shift + enter 向上重开一行

# 编辑器相关

快捷键 说明
Command + P 打开文件，跳转行
command + shift + F 全局查找
control + shift + F 代码格式化
option + command + 箭头 切换编辑器Tab

# 插件相关

快捷键 说明
Command + Shift + P 插件指令
shift + option + U sync上传
shift + option + D sync下载
